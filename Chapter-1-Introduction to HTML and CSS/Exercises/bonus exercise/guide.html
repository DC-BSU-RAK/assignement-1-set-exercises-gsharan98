HTML & CSS Reference Guide

learn HTML and CSS by looking at examples.

HTML Tags

<p> - Paragraph

This tag allows you to define a block of text. The text is automatically preceded and followed by a gap..

<p>paragraph.</p>

<br> - Line Break

Use the <br> To add a line break to text, use this tag. It creates a new line for the next lines of text..

<p>First line of text<br>Second line of text</p>

<strong> - Bold Text

Using this tag, regular text can be made bold..

<strong>bold text </strong>

<em> - Italic Text

This tag is used to make normal text into italic.

<em>italic text </em>

<big> - Big Text

Use the <big> tag used to increase the size of the text.

<big>bigger text  than normal text.</big>

<small> - Small Text

Use the <small> tag used to decrease the size of the text.

<small> smaller text  than normal text.</small>

<pre> - Preformatted Text

The <pre> Tags are used to show text that has been formatted. (spaces and line breaks).

<pre>This is preformatted text that  preserves spaces and line breaks.</pre>

<sub> - Subscript

The <sub> this tag is used to display text as subscript.

<sub>This is subscript text</sub>

<sup> - Superscript

The <sup> this tag is used to display text as superscript.

<sup>This is superscript text</sup>

<!-- --> - Comment

HTML comments are added using the <!-- --> syntax. They are not visible in the final outcone.

<!-- This is a comment -->

CSS Properties

font-size

size of the font.

p {
    font-size: 16px;
}

color

changes the color of the text.

p {
    color: #fff;
}

background-color

sets a background color.

div {
    background-color: #444;
}

margin

Sets any kind of margin around an element.

p {
    margin: 20px;
}

padding

Sets padding inside of an element.

p {
    padding: 10px;
}

border

Defines the border around any element.

div {
    border: 1px solid #666;
}

font-family

Defines the font used for text.

body {
    font-family: 'Arial', sans-serif;
}

text-align

Aligns text within an element.

h1 {
    text-align: center;
}
